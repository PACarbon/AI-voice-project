# 开发板绘制

## 作业绘制法国国旗代码实现
```c
#include <stdio.h>
#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>
#include <unistd.h>

int main(int argc, char const *argv[])
{
    // 1.打开lcd屏幕
    int fd = open("/dev/fb0", O_RDWR);

    if (fd == -1)
    {
        perror("open /dev/fb0 failed!");
        // 打开失败退出主程序
        return -1;
    }

    // 2. 准备渲染在屏幕上数组
    int buf[480][800];

    // 3. 设置二维数组的各个像素的颜色
    /*
    for (int y = 0; y < 480; y++)
    {
        for (int x = 0; x < 800; x++)
        {

            if (x <= 266) // 法国国旗蓝色
            {
                buf[y][x] = 0xff3f48cc;
            }
            else if (x <= 2 * 266)
            {
                buf[y][x] = 0xffffffff;
            }
            else
            {
                buf[y][x] = 0xffb70000;
            }
        }
    }
    */
    // 优化
    for (int y = 0; y < 480; y++)
    {
        for (int x = 0; x < 800; x++)
        {
            int color = 0xffb70000; // 深红

            if (x <= 266) // 法国国旗蓝色
            {
                color = 0xff3f48cc;
            }
            else if (x <= 2 * 266) // 法国国旗白色
            {
                color = 0xffffffff;
            }

            buf[y][x] = color;
        }
    }

    // 将颜色数据写入到lcd屏幕中
    int ret = write(fd, buf, 800 * 480 * 4);
    if (ret == -1)
    {
        perror("write buf failed!");
    }
    
    // 关闭文件描述符
    close(fd);

    return 0;
}
```

**思考**：上面代码中我们的数组 `int buf[480][800];` 是二维数组，若我想使用一维数组 `int buf[480 * 800];`实现，应该如何实现？

**答案**: 二维数组 y，x坐标 转换为一维数组 index 公式为 `index = y * 800 + x`

## mmap函数

**概念**：内存映射（memory map）用于将文件或设备映射到进程虚拟地址空间，实现内存级文件 I/O，以指针直接读写文件，减少内核态 / 用户态拷贝并支持高效共享内存。

```c
#include <sys/mman.h>

void *mmap(void *addr, size_t length, int prot, int flags, int fd, off_t offset);
```

**参数**
- addr：建议映射起始地址；通常传 NULL，由内核自动分配。
- length：映射字节数；必须按页对齐，不足时向上对齐到页大小（如 4KB）。
- prot：	映射区权限：PROT_NONE/PROT_READ/PROT_WRITE/PROT_EXEC（可组合）。
- flags：	映射属性：MAP_SHARED（修改同步文件，多进程共享）/MAP_PRIVATE（写时复制，不回文件）。
- fd： 文件描述符；传 -1 配合 MAP_ANONYMOUS 做匿名共享内存（Linux 常用）。
- offset： 文件偏移量；必须页对齐，通常为 0（从文件头映射）。

**返回值**： 
1. 成功：文件内存映射后的指针地址
2. 失败：`（(void *)-1）`

### 解除映射：munmap
```c
int munmap(void *addr, size_t length);
```
**成功**：返回 0；失败：返回 -1。
**要点**：addr 必须是 mmap 返回的原始地址，length 向上对齐到页大小；访问已解除的地址会触发 SIGSEGV。
