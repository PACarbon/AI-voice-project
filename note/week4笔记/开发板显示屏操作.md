# 文件操作

Linux下一切皆文件。在控制6818开发板时本质上就是操作对应的设备文件,文件的操作步骤如下：
1. 使用`open`方法打开对应的设备文件：以lcd显示屏为例，显示屏的文件路径是 `/dev/fb0`

```c

#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>
int open(const char *pathname, int flags);

// 参数一： 设备文件路径
// 参数二： 权限 O_RDONLY 只读, O_WRONLY 只写, or O_RDWR 读写
// 返回值： 设备文件描述符（后续文件操作都需要依赖文件描述符）
int fd = open("/dev/fb0", O_RDWR);
```
2. 操作：
    - 读： 读取设备内的内容
    - 写： 像设备写入内容
```c
#include <unistd.h>
// 参数一：fd     open函数成功打开文件后的文件描述符
// 参数二：buf    读取出来的数据存放的缓冲区的地址
// 参数三：count  读取多少个字节的数据
// 返回值：
// 成功：成功读取到的字节数（0表示读到文件末尾）
// 失败：-1
ssize_t read(int fd, void *buf, size_t count);

// 参数一：fd  open函数成功打开文件后的文件描述符
// 参数二：buf  要写入的数据存放的缓冲区地址
// 参数三：count  要写入的字节数
// 返回值：
// 成功：写入的字节数（0表示没有东西被写入）
// 失败：-1
ssize_t  write(int  fd,  const  void  *buf,  size_t count);


```
